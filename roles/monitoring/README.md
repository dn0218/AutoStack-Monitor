# Role: monitoring

## 中文说明
### 功能概述
部署监控系统，实现对容器运行状态的可视化。
* **Zabbix Agent 2**: 部署为容器，并映射宿主机 Podman Socket 以采集指标 。
* **Grafana**: 仅在 `node1` 部署，并预装 Zabbix 插件 。
* **演示优化**: 开启匿名登录模式（Admin 权限），方便演示环境即时查看仪表盘 。

### 为什么这样写？
* **Socket Mapping**: 将 `/run/user/.../podman.sock` 映射进容器是 rootless 监控的关键 。
* **Flush Handlers**: 在启动服务前强制执行 `flush_handlers`，确保 systemd 单元文件已由 Quadlet 成功生成 。

---

## English Description
### Overview
Deploys the monitoring system to provide visualization of container operational status.
* **Zabbix Agent 2**: Deployed as a container with the host's Podman Socket mapped to collect metrics.
* **Grafana**: Deployed only on `node1` with the Zabbix plugin pre-installed.
* **Demo Optimization**: Enables anonymous login (Admin privileges) to facilitate immediate dashboard access in demo environments.

### Rationale
* **Socket Mapping**: Mapping `/run/user/.../podman.sock` into the container is key for rootless monitoring.
* **Flush Handlers**: Forcing `flush_handlers` before starting services ensures that systemd unit files have been successfully generated by Quadlet.
